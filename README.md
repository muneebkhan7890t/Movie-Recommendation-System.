AI Recommendation Logic System

This project is a simple recommendation engine built using Python.

The system takes user preferences as input and recommends movies based on genre similarity. The recommendation process uses pattern matching and similarity logic to find relevant items.

Features:
- User preference input
- Genre-based recommendations
- Similarity matching
- Simple and beginner-friendly AI logic

Technologies:
- Python
- Pandas
- NumPy

This project demonstrates the fundamentals of recommendation systems and personalized content suggestions.
